# Custom Recipe Override (Minecraft 1.20.1)

## Overview

**Custom Recipe Override** is a lightweight utility mod for Minecraft Forge 1.20.1 that allows modification of recipes from mods that use custom workbenches.

Instead of being limited to vanilla datapack systems, this mod provides a way to **override and redefine crafting recipes** used by other mods.

---

## Purpose

Many mods introduce their own crafting systems and workbenches, making it difficult to modify recipes using standard Minecraft tools.

This mod solves that by allowing:

* Recipe overrides
* Custom recipe definitions
* Better control over modpack balance

---

### 🔧 Recipe Override System

* Modify recipes from mods with custom crafting tables

### ⚙️ Lightweight Design

## How It Works

* The mod hooks into custom crafting systems
* Existing recipes can be:

  * Replaced
  * Modified
* Changes are applied at runtime

*(Exact implementation may depend on the target mod’s crafting system)*
